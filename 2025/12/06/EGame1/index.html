电子游戏的回忆（一）小学篇

本文最后更新于：2025年12月6日 晚上

我一直想写一些自己从小到大的回忆，写给自己，也是对时代的一个记录。于是从最熟悉的电子游戏方面开始写起，不知不觉就写了4000多字（刚写到小学高年级）。全文都是个人的经历，一定会有一些疏漏和主观看法，请多包涵。话不多说，开始吧。

游戏，我要玩游戏！

学习机

我的童年是从千禧年开始的，最早接触到的电子游戏是家里的“红白机”。

这台“红白机”想必是小霸王，机身是红白配色，通过三根红黄白色的AV线连接客厅的电视。手柄是棕色的长方形，左边是十字方向键，右边是AB两个按键。这台游戏机是我从记事起就有的，可能比我年龄还大。小时候坐在电视机前，小板凳上，和父母一起玩一些超级玛丽，魂斗罗，坦克大战之类的游戏。

过了不久，大概是我四，五岁的时候，有一天我爸回家，提着一个紫红色的盒子，上面印着一个白色的键盘。我一开始以为这就是传说中的电脑，后来才知道这依然是插卡的游戏机。现在回想起来应该是小霸王学习机之类的产品。白色的键盘上印着五笔输入法的字形，键盘上方正中间有一个游戏卡的插槽。盒子里除了两个更先进的手柄，还有一把手枪，可以用来玩一个打鸟的游戏。

在网上找到一个类似的图，大概长这样：

越看越像我记忆里的样子，甚至对红蓝白色的游戏卡都有印象。

因为我比较小，游戏卡带都是父母买的，他们买啥我玩啥。所以我至今都不知道这些卡带是在哪买的，也没有和同龄人一起玩过。印象比较深的游戏除了上面提到的，还有马戏团，冒险岛，炸弹人，淘金者，打砖块，赛车，影子传说这些。

我们的游戏水平都很一般，魂斗罗一代可以调30条命，能打通关。魂斗罗二代不会调，最多只能打到第三关。我妈玩炸弹人和淘金者都很老练。我比较爱玩打砖块，马戏团这类难度不太高的。

后来发现坦克大战和淘金者都可以自定义地图，找到了画地图的乐趣。

与电脑的初次接触

大约是我上小学二年级的时候，家里买了电脑。

主板是七彩虹，内存512MB，硬盘60GB，显示器是一台窄屏的液晶显示器，在那时候应该比较流行。根据AI推测，这是2006到2007年市场上入门款的主流配置。同时配了电脑桌和音箱。我爸从电脑城带回来了三本书，只记得其中一本书名是《21天学会上网》，剩下一本是基础操作，一本是 WindowsXP 系统指南。

说回游戏，首先接触的当然是系统自带的小游戏，三维弹球，纸牌，蜘蛛纸牌，空当接龙。还记得最上面的几个（红心大战）似乎是需要联网的，怎么也玩不了。

除此以外，不知道是亲戚还是装机师傅，给我顺便装了一些单机游戏。

空中霸王2 海湾战争
飞行射击类的游戏。画面和背景音乐都特别炫酷，操控飞机躲子弹，发射导弹，打起来很爽。在这个游戏里第一次知道地上画着一个圈，圈里一个H，是飞机降落的地方。

游戏视频: https://b23.tv/dQwdJ2z

重装兵器
也是一个战斗的。这次操作的是坦克。脑中自然浮现“贝塔贝塔贝塔，开坦克的贝塔…”。今天才发现是宝开公司制作的。

游戏视频: https://b23.tv/PR1L3b8

美女餐厅
这个游戏我和父母都爱玩。鼠标操作，主角是一个餐厅女老板，需要招呼客人，点菜，上菜，收拾桌子。客人等的时间久了就会生气。随着关卡发展，主角的餐厅越来越大，难度越来越高，比较考验手速。还有一个技巧，就是把穿着不同颜色衣服的顾客放到对应颜色的座位上，有加分。

大富翁
经典游戏，回想起来就是浓浓的台湾腔。“又到了每月乐透开奖时间…奖金将累积到下月，希望下一个得奖者就是你。”

游戏内容还是很丰富有趣的，各种卡片道具魔法屋，福神财神衰神，贷款融资股票…可选的角色也很多，每个角色的配音都非常有特色。这应该是我玩的第一个“热座模式游戏”，跟亲戚的小孩轮流坐到电脑前操作自己的回合。

印象比较深的是卡片里有一张“嫁祸卡”，当时根本不懂什么是嫁祸，甚至不太认识祸字，只认识嫁，以为是结婚的意思。后来发现不是这么回事。

抗日 血战上海滩
这应该是我接触的第一个FPS游戏。手里只有一把手枪，做一个孤胆英雄。游戏的第一关叫牛刀小试，那会不明白这个成语，一直读作牛试小刀，感觉这样念起来比较顺。

红色警戒2
一开始不知道为什么装的是个英文版，不会 Load 和 Save，只能开启新战役，所以每一次都是从训练关开始打，点击操控单位走来走去，射击打靶，对于二年级的我来说已经足够有趣了。后来有一次打到了训练关的第二关，发现可以变形的基地车，可以造发电厂，就特别激动。再后来，装上了中文版，也知道了存储和载入，就之后再讲吧。

QQ游戏
很多人接触网络对战游戏都是从这里开始的吧，甚至接触QQ游戏比QQ还早。刚装好电脑，亲戚就给注册了一个QQ号，设置成自动登录，昵称叫作1231，就可以让我妈愉快地打挖坑了。玩的时候，经常看到别人的昵称也叫123,1231之类的。

后来等我真正注册QQ号的时候，这个1231的账号早都不知道丢在哪了。

QQ音速
很奇妙，我感觉我玩QQ音速是特别早的，但一丁点细节都记不住了。只记得画面很炫，按方向键来左右上下，类似后来的神庙逃亡。可能是我接触的第一款网游以及音游？

跑跑卡丁车
唯一真神。

陪伴了我整个少年时代的网络游戏（2007年到2012年）。对一个小朋友来说，吸引我的除了随着转弯自己也在椅子上左右摆动的沉浸感，还有各种各样主题的地图，每种主题的背景音乐都不相同。尤其是太空类型的地图，颜色鲜艳，通常都会有各种跳跃和加速装置，小时候特别喜欢玩。太空基地，太空喷射滑坡，太空跳跃峡谷。我到现在还记得有一天回到家里，发现更新了新的地图，叫做太空极限跳跃，有连续三个跳跃平台，很带感。根据游戏新闻，那是2007年8月30日。

全新赛道“太空极限跳跃”体验跳跃无极限_52pk跑跑卡丁车官方新闻_popkart.52pk.com

我当时在电信一区，用户名是 lionoctopus，狮子八爪鱼，是我自己起的，八爪鱼的英文来自一本台湾出的小朋友学英语的书。密码到现在还是我家用的最多的密码，那是第一次使用。昵称是皮小六，也是从小到大注册各种账号用的最多的昵称，第一次使用。

当然，小孩玩这种游戏只能不断撞墙，所以一开始完全是打道具赛和故事模式的，玩不了一点竞速。所以考L3驾照的最后一关（城镇高速公路的计时赛）过不去，最后是请邻居家的哥哥帮我过的。后来发现，这个计时赛只要撞墙次数少一点就能过，根本不需要漂移集气。只能说是我撞墙太多了。

森林木桶，森林蘑菇，城镇钟楼，冰山急速（忘了是不是这个名字），太空喷射滑坡，到后来工厂地图，龙行华夏地图，想到每种主题，以及主菜单，房间，商店，小屋的界面，脑内都能自动播放它的BGM。每到圣诞节的时候，冰雪主题的赛道背景音乐都会换成圣诞歌，有非常欢乐的节日氛围。

【【怀旧向】旧版《跑跑卡丁车》背景音乐合集-哔哩哔哩】 https://b23.tv/bkNqEfz

还有更多与跑跑有关的故事，我们之后再说。现在登陆账号还能看到2007-2012年获得的道具：

“拥有穿梭时光的魔力……该道具将于2007年12月5日后自动消失”

熟练地玩电脑

大约到了2008年的时候，我已经能够熟练地玩电脑和上网冲浪了。

Flash小游戏
那会大概每天都在玩4399之类的Flash小游戏。印象深的除了经典的黄金矿工，还有就是玩美少女换装类游戏被人发现。(?)

奥运会/世博会
2008年北京奥运会和2010年上海世博会，都有一些官方的小游戏。现在只记得一个水立方的游戏，画面是水立方外表一样不规则形状的格子，点击格子可以吸收或者填上水的颜色，益智类涂色的那种。

世博会就是在网上参观虚拟场馆，和街景地图一样，每走一步给你介绍展品。还挺有意思。

这两个我不太记得是哪个必须安装 Java 才能玩，反正这是我第一次知道 Java （和它的咖啡图标），比在我爸手机上安装 Java 游戏还要早。

摩尔庄园/赛尔号
说起Flash游戏，这俩也是flash.

我能背过的互联网账号可能只有我的QQ号和米米号了吧，今天试了试，居然登录成功。

摩尔庄园大概是08年左右玩的。换上各种职业的衣服，玩各种小游戏。记得主线剧情还有什么狸猫换太子的桥段。

布置小屋，小屋里面可以摆放一个纪念装置，里面有每个月收集的小球。我是2009年全部集齐了，2010年只有两个脚印，看来已经不太沉迷了。

赛尔号是6月份公测，我是8月底开始玩的。这个好像是后来有和同学一起玩。一开始精灵什么的都比较简单，每天中午放学回家吃完饭，不睡午觉，直接进入赛尔号做每日任务，直到下午上学。印象比较深的是后来出了一个独眼的boss（大概叫卡纳斯什么的），根本打不过，很无奈。

后来把我的号借给同学，他给充了很久的超能nono，让我体验了一把付费玩家的感觉。我好像也进行过一次充值，当时的步骤是在报刊亭买点卡，在游戏里输入，就跟充Q币一样。但后来这个号几乎是给同学玩了，所以我对它的感情也不怎么深。

QQ游戏
正式注册了QQ之后，开始玩连连看，对对碰，大家来找茬。这些游戏都玩过吧？反正我记得把能体验的游戏都体验了一遍（主要是棋类），后来和同学开语音玩飞行棋，也留下过快乐的印象。

QQ三国
时间倒回一些，说一说正经的网游。比如QQ三国。这是一个横版的RPG，可以跳上跳下。选择不同职业，做任务，打怪升级学技能。可能是我第一次玩RPG类型游戏。


蜀国发每日任务的NPC是法正，经常让我给他找本《论语》垫桌子腿。这个任务印象很深。

QQ三国的背景音乐也很好听。突然发现我喜欢的游戏都有非常不错的背景音乐。

【游戏BGM选集【QQ三国】（53P）】 https://www.bilibili.com/video/BV16W411d78U/?p=4&share_source=copy_web&vd_source=4bbb75289d26a8a7c2b0a5c3d6678781

第一次跟同学开着语音打游戏可能就是QQ三国。

那时候为了点亮QQ的图标，其他的腾讯系的网游也都有些尝试，比如QQ自由幻想，这是一个45度俯视的2D RPG，和QQ幻想还不是一个游戏，但玩不太明白。

QQ堂倒是和同学一起玩过不少次，同学比我厉害太多了，抢包子整天被虐。所以对这游戏既爱又恨。

QQ炫舞…只记得楼下理发店的小哥整天在玩，没有接触过。

继续打红警2
终于，在安装了中文版并且学会了存档/载入之后，可以愉快地打战役了，然后就卡在了盟军第五关暗夜，一上来只能操控三个间谍和一个谭雅，根本不知道/不会用间谍。后来还是请表哥来我家指导，才明白怎么伪装成敌人，躲着狗走，进入发电厂断电。（之前我一直以为从直升机下来的三个间谍是我第三关从白宫里救出来的总统）。

战役都打通关后，有一次看《小哥白尼》军事版，里面提到红警2里面的盟军多功能步兵车，放入不同兵种可以有不同的效果，比如放工程师能维修，放尤里新兵能喷火。

《小哥白尼》是面向小学生发行的科普杂志，大概是月刊，好像有科学和军事两版？在学校订的，每次订购还送小玩具，比如带手柄拉线的竹蜻蜓。小时候一直以为名字是小哥/白尼，因为杂志封面上小哥写在第一行，白尼写在第二行。那会儿还不知道哥白尼。

于是我就不解了，尤里新兵是什么东西？之后打开游戏试验了好几次，把尤里放进去，只能哗啦哗啦的发水波，没看见喷火啊。

直到后来有一次在海底捞吃饭（玩电脑）才知道是怎么回事。那时候海底捞会在大厅放一排电脑，大概三五台，吃饭前（中途）可以去玩。（当时海底捞还不太出名，我对它的印象就是扯面和电脑。。。）于是第一次玩到了红警2的资料片《尤里的复仇》，回家赶紧下载下来玩，原来尤里新兵出自这里，还有新的战役可以打，的确比原版好玩。

再后来，了解到各种修改版的MOD。当时在网上不知道从哪里找到了一个人，给我分享这些资源。现在还能找到他发来的邮件。

还下载过一个合集，里面大概有十几种各个版本的MOD，包括什么兵临城下，心灵终结，蓝色警戒。

PSP

大约是三年级的暑假，我从亲戚家里借来了PSP，开启了短暂的掌机游戏生涯。当时接触到的游戏有这几个：

索尼克
Sonic Rivals, 经典赛跑游戏。能记得在菜单选项里清脆的点击音。当时觉着很帅，跑起来很爽。

乐克乐克
这是当时最喜欢的游戏，闯关类型，吃桃子可以增加小球数量，小球可以分裂合体。有各种玩法，各种隐藏收集要素，每次都想收集20个球。小球一起唱歌，真可爱。

后来才知道我控制的是星球，让星球左右倾斜和震动。（？）

写的时候又去看了看游戏流程视频，太好玩了，每个人都值得去玩。

【乐克乐克之歌】 https://www.bilibili.com/video/BV1Fs411B76J/?share_source=copy_web&vd_source=4bbb75289d26a8a7c2b0a5c3d6678781

真三国无双4 PSP移植版
实际上叫做《真·三国无双 2nd Evolution》。这是我最早接触的三国无双系列。而且PSP上的这版三国无双也是比较特殊的。因为它是由战棋式的大地图和实际操控人物战斗（割草）的小地图组成的。每个小地图是战棋里的一个格子。地图上的砦可以让杀死的将领复活，粮草库可以增加可用时间。每次打虎牢关的时候都很艰险，眼看着吕布一个格子一个格子飞快地往袁绍的大本营打过去，这个时候只能拼速度尽快去打董卓。

后来玩了大地图的三国无双，我感觉这种走格子模式还挺有意思的，有些怀念。

继续玩电脑

真三国无双系列
接着上文，当时痴迷上了真三国无双，在电脑上把第三代，第四代，第五代甚至无双大蛇都下载下来，玩得不亦乐乎。细节方面已经记不住什么了，只记得第五代(简称355)是比较特别的。

还玩到了“敌羞，我去脱他衣”的空耳汉化版。好像是3dm的汉化？

但很奇怪，那会痴迷于三国无双系列，还找院子里的小伙伴到我家一起玩，但我一直没有尝试过光荣公司更出名的《三国志》系列。可能还没有到玩回合制策略游戏的年龄。

红警3
2008年，听说EA要出红警3。发售时间大约是10月。总之，我在年底前从表哥那拿到了红警3繁体中文版的安装光盘。

首先印象深刻的是开场电影：苏联即将毁灭，上校查丹科坐上时光机回到了1927年，布鲁塞尔，跟爱因斯坦握手，让他化作一缕青烟，回到现代，苏联即将踏平欧洲，但是遭到帝国入侵，“没有爱因斯坦就没有核武器”。然后武士帽子机甲高达都杀过来了。

虽然当时知道红警2原版的战役也是有过场电影的，但玩的时候谁看过啊！一般下载到的红警2都是200多MB的版本，去掉了所有的过场电影。后来在优酷专门补过这些过场动画。

红警3明显更精致，每一关都有过场电影，都有情报官介绍任务。很有沉浸感，着实是一个亮点。

从操作手感来说，一开始的确会不习惯鼠标左键选中，右键行动，固定矿位。后来慢慢就习惯了。

从玩法来说，3代更突出兵种克制关系，战场迷雾更真实，更需要侦查，海军空军大扩充。这些都是为了在玩法上更有竞技性和多样化，不像原来二代都玩憋憋乐，在家里屯了一万只部队再出去打。

另外，游戏里每一个兵种都有详细的视频版单位介绍，在网上还能找到详细的单位设定，每一个单位都有设计图，制造方，装备，背景故事，战场特征，读起来非常带感，非常爱看。

维和轰炸机的设计图


天启坦克的详细单位介绍。

总之，这是我最喜欢的游戏之一。但这时候还只是自己玩。

偷菜
2009年4月，腾讯推出了”QQ农场”。全民偷菜的时代开始了。现在只记得红土地黑土地了。QQ牧场倒是印象更深，因为一开始测试是邀请制的，我为了提前玩上QQ牧场还在百度知道上找到了一个有邀请资格的人，加了QQ，成功收到邀请。

差不多那两年，有一句网络流行语是“神马都是浮云”，于是在QQ牧场里面可以养殖神马收获浮云（？）

2011年春节期间饲养金兔子、种金条树获得作物，88个金元宝/金条可获得一次抽奖机会，有机会抽到神马幼崽

印象更深的其实是另外的QQ空间小游戏，抢车位和Q宠大乱斗。我一直觉着Q宠大乱斗是这几个里最好玩的。

顺带一提，说到QQ宠物，当然也养过，但是没有充值Q币就没有办法养得好。后来好像还出了养小熊？对养小熊还有点印象。

CF
小学高年级那会也是 CF 最流行的时候吧？很多同学都去网吧一起玩。我也在家里一起开语音玩过，水平一般。时间大概是救世主模式推出前后，查到的时间是2010年12月14日。

另外印象深刻的一点是同学在语音里把卡BUG念作卡Biu记，虽然我当时也是第一次听说 bug 这个词，但我感觉绝对不是按字母念作”哔优记”或者”Biu记”的吧。。。

另外还有一些有趣的单机游戏大约也是小升初这段时间玩的，比如植物大战僵尸，机械迷城，整蛊邻居，麦克斯与魔法标记。虽然没有能大书特书的点，但都是陪伴我度过美好时光的好游戏哇！

未完待续。接下来该进入初中了。
